In this project, I explored a dataset that contains information about movies and TV shows available on Netflix. I used Python and the pandas library to load the data, look through it, and better understand what kind of content is available on the platform.



The dataset includes details such as whether a title is a movie or a TV show, the year it was released, its rating, and the country where it was produced. Each row in the dataset represents one title on Netflix.
